Flip Colors

function setColor(name) {
    body.style.backgroundColor = name;
}

function randomColor() {
    let R = Math.round(Math.random() * 255);
    let G = Math.round(Math.random() * 255);
    let B = Math.round(Math.random() * 255);

    theColor = `rgb(${R}, ${G}, ${B})`;

    body.style.backgroundColor = theColor;

    if (R + G + B < 383) {
        body.style.color = "white";
    } else {
        body.style.color = "black";
    }
}

Le code prend le body de la page. Il le met en fond magenta avec setColor("magenta"). La fonction randomColor prépare 3 valeurs : rouge, vert, bleu — chacune avec une valeur aléatoire entre 0 et 255. Ensemble, elles forment une couleur RGB au hasard. Si la somme R+G+B est inférieure à 383, le texte passe en blanc pour rester lisible sur fond sombre — sinon il reste noir.
