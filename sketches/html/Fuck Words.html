Fuckwords (18+)

const Strategies = [
    "Fuck", "Shit", "Bitch",
    "Ass", "Dogshit", "Pussy",
    "Damn", "Bastard", "Bullshit",
    "Wavefuck", "Flowshit",
    "Sparkass", "Edgefuck",
    "Clusterfuck", "Pileofshit",
    "Colorfuck", "Deepfuck",
    // ... 82 mots au total
];

let lastIndex = -1;

function getNewWord() {
    let randomIndex;
    do {
        randomIndex = Math.floor(
          Math.random() * Strategies.length
        );
    } while (
      randomIndex === lastIndex
      && Strategies.length > 1
    );

    lastIndex = randomIndex;
    const newWord = Strategies[randomIndex];

    strategy.style.opacity = "0";
    strategy.style.filter = "blur(4px)";

    setTimeout(() => {
        strategy.innerHTML = newWord;
        strategy.style.opacity = "1";
        strategy.style.filter = "blur(0)";
    }, 150);
}

Générateur de mots aléatoires provocateurs. Un tableau de 82 mots composés mélange insultes classiques et néologismes absurdes (« Clusterfuck », « Sparkass », « Colorfuck »…). À chaque clic sur le bouton, la fonction getNewWord tire un index au hasard — en s'assurant de ne jamais répéter le mot précédent grâce à une boucle do…while. Le mot actuel disparaît avec un fondu + flou (opacity → 0, blur → 4px), puis après 150 ms le nouveau mot apparaît en fondu inverse. Simple, efficace, un peu vulgaire.
