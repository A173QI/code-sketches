Glimmer Observation

function draw() {
    // Easing: circle follows
    // the mouse with delay
    ancienX = ancienX
      + ((mouseX - ancienX) / 100);
    ancienY = ancienY
      + ((mouseY - ancienY) / 100);
    circle(ancienX, ancienY, size);
}

function randomColor() {
    // Random background color
    currentBgColor = random(palette);

    // Different fill color
    // to ensure contrast
    let fillCol;
    do {
        fillCol = random(palette);
    } while (fillCol === currentBgColor);

    background(currentBgColor);
    fill(fillCol);
    stroke(currentBgColor);
    size = random(25, 250);
}

function mouseClicked() {
    randomColor();
}
My Canva

Un cercle suit la souris avec un effet d'inertie (easing). À chaque frame, la position du cercle se rapproche de la souris de 1/100e de la distance restante — ce qui crée un mouvement fluide et en retard.

Le fond n'est pas rafraîchi dans draw(), ce qui laisse une trace du mouvement sur le canvas, comme une traînée de peinture.

En cliquant, la fonction randomColor() choisit aléatoirement une couleur de fond et une couleur de remplissage différente depuis la palette. La boucle do...while garantit le contraste entre les deux couleurs.

La taille du cercle change aussi aléatoirement entre 25 et 250 pixels à chaque clic.
