Canvas Style

for (let y = 0; y < 400; y++) {
    for (let x = 0; x < 400; x++) {

        noStroke();

        // Le rouge augmente vers la droite
        // Le bleu augmente vers le bas
        fill(x * 14, 14, y * 15);

        // square(PosX, PosY, Taille)
        square(x * 12, y * 10, 10);

        print("Case: " + x + "," + y);
    }
}
My Canva
Grille de carrés colorés avec un dégradé calculé mathématiquement. Le rouge augmente horizontalement (x × 14), le bleu verticalement (y × 15). C'est un sketch pédagogique — tout est dessiné dans setup(), pas d'animation. La double boucle for parcourt chaque case de la grille et pose un carré coloré.
