Sine Chain

// Shift positions: each segment
// takes the previous one's place
for (let i = numSegments - 1; i > 0; i--) {
    x[i] = x[i - 1];
    y[i] = y[i - 1];
}

// The head follows the mouse
// with a sine wave undulation
x[0] = mouseX;
y[0] = mouseY + sin(frameCount * 0.15) * 45;

// Draw with decreasing size
// and cycling palette colors
for (let i = 0; i < numSegments; i++) {
    let size = map(i, 0, numSegments, 45, 8);
    let col = color(palette[i % palette.length]);

    fill(col);
    noStroke();
    ellipse(x[i], y[i], size, size);
}
My Canva

Une chaîne de 40 cercles qui suivent la souris en serpentant.

Le premier segment (la « tête ») suit le curseur avec une ondulation sinusoïdale : sin(frameCount * 0.15) * 45 ajoute un mouvement de vague vertical.

Chaque segment suivant prend la position du précédent grâce à une boucle inversée — c'est ce qui crée l'effet de traîne fluide.

La taille décroît progressivement du premier au dernier segment avec map(), et les couleurs cyclent dans la palette grâce à l'opérateur modulo (i % palette.length).
