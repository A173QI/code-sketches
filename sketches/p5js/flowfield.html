Flow Field

function draw() {
    background('#131B23');

    let cols = floor(width / spacing);
    let rows = floor(height / spacing);
    let time = frameCount * 0.008;

    for (let x = 0; x <= cols; x++) {
        for (let y = 0; y <= rows; y++) {
            let px = x * spacing + spacing / 2;
            let py = y * spacing + spacing / 2;

            let d = dist(mouseX, mouseY, px, py);
            let mouseForce =
              map(d, 0, 300, 2, 0, true);

            let n = noise(
              x * 0.08, y * 0.08, time
            );
            let angle = n * TWO_PI
              + mouseForce
              * atan2(mouseY - py, mouseX - px);

            stroke(palette[(x+y) % palette.length]);
            push();
            translate(px, py);
            rotate(angle);
            line(-spacing*0.4, 0, spacing*0.4, 0);
            pop();
        }
    }
}

Champ de vecteurs basé sur le bruit de Perlin. Une grille de petites lignes s'orientent selon le bruit, perturbées par la position de la souris. Plus la souris est proche, plus les vecteurs pointent vers elle — grâce à atan2 et une force mappée avec map(). Les couleurs alternent dans la palette selon la position dans la grille.
