Fractal Tree

function branch(len) {
    // Draw the branch line
    line(0, 0, 0, -len);

    // Move to the end of that branch
    translate(0, -len);

    // Shrink the branch length
    len *= 0.67;

    // Recursive base case:
    // stop when length is too small
    if (len > 6) {
        push();
        rotate(angle);  // Rotate right
        branch(len);     // Draw branch
        pop();

        push();
        rotate(-angle); // Rotate left
        branch(len);    // Draw branch
        pop();
    }
}
My Canva

Un arbre fractal récursif dessiné depuis le bas du canvas. La position horizontale de la souris contrôle l'angle de ramification.

Chaque branche se divise en deux sous-branches avec une longueur réduite de 33% (len *= 0.67). La fonction branch() s'appelle elle-même deux fois : une fois avec rotate(angle) et une fois avec rotate(-angle).

La récursion s'arrête quand la longueur devient inférieure à 6 pixels — c'est le cas de base qui empêche une boucle infinie.

push() et pop() sauvegardent et restaurent l'état de transformation, pour que chaque branche parte du bon endroit.
